Installation

1. Install Node.js and NPM : Download Node.js Recommended Version from https://nodejs.org/en/download/ and install it on your system. NPM comes bundled with Node.js

2. Install Dependency : Make sure you install all dependencies first, First run npm -i or run yarn (shorthand for npm install)

                    npm i 
                    or
                    yarn

To use configured development mode simply run-- 

                    npm run start
                    or
                    yarn start

To build all files for the production run --


                    npm run build
Note: An output is minified via --mode flag which is set to production (already minified, optimized bundle) - additionally dist directory will be removed every time when you run this script to make sure there are no artifacts left in the output location directory.

Export files with development mode (output is not minimized/optimized). To use it run 


                    npm run dev

How to change content

Open Component > layout or page > and edit.
